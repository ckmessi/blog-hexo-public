Windows/Ubuntu/Mac OS查看文件md5值的命令

MD5算法的一项重要功能就是一致性验证。

有时候，为了判断下载文件是否正确，可以通过判断本地文件的md5值是否与服务器文件的md5值相同完成。

这里总结在三种不同平台下，计算文件md5值的命令如下。

Windows

使用certutil命令计算文件的哈希值，可以选择MD5、SHA1或SHA256算法。

Ubuntu

在 Ubuntu 16.04 操作系统中可以使用 md5sum 命令。

Mac OS

在 Mac OS 操作系统中可以使用 md5 命令。

下载文件后，顺手比对一下文件的md5值是一个好习惯。

一方面确保下载数据完整，另一方面防止中间人攻击。
