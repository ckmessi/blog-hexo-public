Vue导出HTML页面至PDF

背景

领导说最近写 PPT 写得有点累，很多实验结论贴图容易弄混，问我有没办法直接写代码把相关素材导出成 PDF。

领导问，当然没办法也得创造办法啊。

上网查了一下，把 html 元素导出为 PDF 的方案确实有，主要需要使用到 html2canvas 和 jsPDF 这两个库。

比如一共有 4 页 1920x1080 的页面内容，在打印第二页的时候，应该从 -1080px 位置开始打印，这样显示出来的正好是第 2 页应该有的内容；同理，第三页应该从 -2160px 位置开始打印。

导出功能页面

在导出功能页面中，我们需要添加一个按钮，完成点击后下载 PDF 文件的功能：

需要注意的是，在这个页面中我们定义了三个重要的变量：

  • pdfDomName：标识 PDF 需要打印的 DOM 元素
  • pdfPageWidht：标识 PDF 打印的单页宽度，以像素为单位
  • pdfPageHeight：标识 PDF 打印的单页高度，以像素为单位

目前网上查到的相关方案基本是直接打印一个 A4 纸大小的页面。实际上在不同的使用场景，可能更多的是希望打印出类似 PPT 制作的页面，可能是 16:9 或 4:3 比例的页面内容。可以通过修改这里的参数进行设置。
