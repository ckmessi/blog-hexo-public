bpy的安装和配置

背景

bpy 是 Blender 官方配套的 Python API 工具包，通过使用 bpy 能够通过 Python 代码实现 Blender 的一些功能。

安装

首先创建一个 Python3.7 的环境，注意一定得是 3.7 版本，因为 bpy 包声明了 >=3.7, <3.8 的限制：

直接通过 pip 安装 bpy 包：

与其它 Python 包不同，在安装完之后，虽然写着 Successfully installed，但是如果现在直接 import bpy，会抛出错误：

在 bpy 的 PyPI仓库主页 提到，在安装之后要执行 bpy_post_install：

然后再导入包时上面的错误就消失了，但有一个奇怪的点，bpy 明明依赖于 numpy，却在安装的时候没有自动安装好，导致还有问题。

执行 pip install numpy 后，才算真正完成了 bpy 包的安装。
