假定 [projectName] 名称为 mysite，执行成功后，会在当前目录生成一个 mysite 的目录，结构如下：

其中里面那个 mysite 是一个包，下面放了一些 Web 服务器开发常用的文件，比如 urls.py、settings.py 等；manage.py 则是一个入口，用于启动项目。

这些文件的含义可以参考官方文档：

  • manage.py：一个命令行工具，用于与 Django 项目交互；
  • mysite/__init__.py：空文件，告诉Python这个目录是一个 Python package；
  • mysite/settings.py：项目的配置信息；
  • mysite/urls.py：URL声明，相当于网站的入口目录；
  • mysite/asgi.py：对于 ASGI 兼容的 Web 服务器的入口；
  • mysite/wsgi.py：对于 WSGI 兼容的 Web 服务器的入口；
