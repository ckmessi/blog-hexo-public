初识Flask

背景

Flask 是一个使用 Python 撰写的轻量级 Web 应用框架，基于 Werkzeug WSGI 工具箱和 Jinja2 模板引擎。

安装

安装 Flask 将自动安装以下依赖：

Debug模式

在开发过程中，可能不希望反复地重启服务（PHP就没有这个问题！），Flask 提供了一种自动 reload 的功能，这样当代码发生改动时，服务器将自动重新载入。
同时，这种模式也给提供大量有用的调试信息。

添加 FLASK_ENV=development 实质上做了三件事：

  • 激活了调试器；
  • 激活了自动重载工具；
  • 开启 Flask 应用内部的 debug 模式；
