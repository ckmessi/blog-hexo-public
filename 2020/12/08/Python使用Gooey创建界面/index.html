Python使用Gooey创建界面

背景

Gooey 是一个库，能够将几乎所有 Python 代码转换成带 GUI （图形操作界面）的程序。
使用 Gooey 的方式很简单，在方法中使用 @gooey 注解即可。

环境基础

由于使用的界面相关的库，在启动时可能会抛错：

因为 MacOS 如果要启动 Python 的 GUI 程序，需要使用 pythonw 命令启动。
注意 MacOS 的 pythonw 可能是在默认路径 /usr/bin 中，版本与当前使用的 Python 虚拟环境可能不符合。可以在 conda 环境中使用 conda install python.app 来修复这个问题，参考 这篇文章 解决。
然后就可以使用 pythonw [python_script_file] 语句执行了。
