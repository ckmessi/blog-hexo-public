初识faiss

背景

由 Facebook AI Research 开发的fassi 是一个针对密集向量的高效的相似性查询和聚类库，它包含了在一个向量集合中操作的各种算法，甚至于在内存中不可行的方案。它包含了评估的代码以及参数的调整策略。fassi 是用 C++ 撰写的，同时兼容 Python/numpy。一些最常用的算法也在 GPU 上实现了。
更多信息可以参考其 代码仓库。

使用示例

通常使用 faiss 的场景是大规模数据的索引与查询，假定有一个 gallery 和一个 query，目标是在 gallery 中查找 query 的 k 近邻。
规范化定理符号 d 表示特征向量维度， g_array 表示一个 num_g x d 大小的矩阵，表示 num_g 条底库特征；q_array 表示一个 num_q x d 大小的矩阵，表示 num_q 条底库特征。

构建索引

为查询目标（底库）构建索引：

Faiss 基于 Index 对象 构建，它能将数据库的向量集合封装好进行预处理，以获得更优的查询性能。有各类索引可以使用，这里使用了最简单的版本 IndexFlatL2，执行暴力搜索L2距离。
在构建索引的时候，需要明确声明向量的维度大小，其它索引可能需要一个训练过程，这里 IndexFlatL2 可以忽略。
当索引构建完成后，有两个操作可以执行：add 和 search。其中 add 能够把元素添加到索引中，比如将待查询的 gallery 数组添加进去。而 search 则用于检索和查询。
