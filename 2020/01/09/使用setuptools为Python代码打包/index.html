使用setuptools为Python代码打包

背景

如果希望将自己的 Python 代码发布出去，让别人能够方便地使用，通常会将 Python 代码打包，并上传到 PyPI 或其它私有仓库中。
这里简单介绍如何使用 setuptools 对 Python 代码执行打包操作。

Python 代码的打包和发布有一套完整的流程和机制，这里仅介绍最简单的一种方式，分为两步：

  1. 将 Python 代码打包成一个压缩文件
  2. 使用 pip install 安装
  3. 引用库文件
