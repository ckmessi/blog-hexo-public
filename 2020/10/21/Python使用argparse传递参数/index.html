Python使用argparse传递参数

背景

argparse 是在撰写 Python 代码时最常用的一个库，它能够方便地解析命令行参数。使用 argparse 而不是直接将一些输入参数定义在代码中，能够避免在版本库中引入一些路径相关的敏感信息。

参数主要选项

参数名称

不带 - 前缀符号的是位置参数，在使用时不需要声明参数名称，输入的值会自动被解析到对应位置的参数变量中。
带 - 和 -- 前缀符号的是可选参数，在使用时需要使用 [变量名称] [变量值] 或 [变量名称]=[变量值] 的方式。

参数类型

对于 add_argument() 方法，有关键字参数 type，由于在命令行输入时默认都是字符串类型，可以通过该方法指定输入参数的类型。

默认值

通过 default 关键字参数，可以设定未输入该变量时的默认值。

参数解释

通过 help 关键字参数，可以设定该参数的含义解释。

其它

  • 使用 choices 可以给出参数的候选值列表；
  • 使用 required 声明这是否为必须给定的参数；
  • 使用 dest 相关于把该位置参数关联到一个特定的名字，可以灵活控制；
  • 使用 action 指定操作，默认是 store 表示存储，常用的还有用 store_true 表示布尔型参数的操作；
  • 使用 nargs 灵活控制这个参数消耗的命令行参数个数。
