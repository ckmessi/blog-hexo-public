React Router v4的使用

最近在入门React，学习的过程中发现一些样例的Router不好使用。经过摸索后发现，react-router版本不是向下兼容的，react-router v2和v4有着截然不同的使用方法。网上许多样例都使用的是react-router@2.，因此在依赖包为react-router@4.时无法正确运行。

根据react-router@4.*的一些参考教程，简单描述撰写一个能够使用的React Router的步骤。

首先要说明的是，React路由相关的组件有许多，包括react-router，react-router-dom和react-dom。在这种情况下，应该使用npm安装react-router-dom。

You should almost never have to install react-router directly. That package provides the core routing components and functions for React Router applications. The other two provide environment specific (browser and react-native) components, but they both also re-export all of react-router's exports.
