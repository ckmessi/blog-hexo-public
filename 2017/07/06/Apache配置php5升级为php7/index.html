Apache配置php5升级为php7

最近遇到一个问题，在测试服上测通了的代码，推到正式服之后却出现莫名其妙的问题。后来经多方查找，终于发现原来是正式服的php版本和测试服的不一致。测试服是php7，而正式服是php5.6。
这样就有了升级php版本的需求。
