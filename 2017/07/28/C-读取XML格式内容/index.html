C#读取XML格式内容

XML(Extensible Markup Language)可扩展标记语言是一种标准通用语言的子集，通常我们将其称为XML格式，对应的还有JSON(JavaScript Object Notation)JS对象标记数据格式等。
在C#中要读取XML格式的数据，常用方法是按照XML语法对数据进行解析，然后提取感兴趣的信息。

使用XmlReader进行流式读取的方式

有些时候，当整个XML数据太大时，使用XmlDocument进行数据的载入.load会抛出out of memory的异常。
这时候可选方案是使用XmlReader类，以流式读取XML格式字符串内容，一边读取一边处理。

当然，使用流式读取的一个缺点在于，无法即时地对xmlStr数据本身进行编辑操作。通常只能在读取的过程中将相关信息记录下来，在读取结束后执行对应的编辑。
