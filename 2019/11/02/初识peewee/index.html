初识peewee

简介

Peewee 是一个简单轻量级的ORM，它的概念很少，但表达能力很强，学习和使用它非常简单直观。

  • 轻量级的功能丰富的ORM
  • 支持 Python 2.7+ 和 3.4+
  • 支持 sqlite，mysql 和 postgrasql
  • 有丰富的扩展

查询操作

使用 ORM 的意义在于不用重复撰写繁杂的 SQL 语句，可以直接使用 ORM 提供的方式完成数据在查询。
Peewee 提供的接口很简单，符合正常的人类思维。

  • Model.create()，执行 INSERT 查询；
  • Model.save() 和 Model.update()，执行 UPDATE 查询；
  • Model.delete_instance() 和 Model.delete()，执行 DELETE 查询；
  • Model.select()，执行 SELECT 查询。

总结

Peewee 是一款简单轻量级的 ORM，在 Python 项目中可以方便地使用它进行数据库操作。
Peewee 可以理解为脱离了 Laravel 框架的 Eloquent，封装了数据库操作相关的大多数接口，调用方式简洁易懂，能够减少许多撰写繁杂 SQL 语句的工作量。
