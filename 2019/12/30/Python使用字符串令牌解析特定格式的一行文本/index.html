Python使用字符串令牌解析特定格式的一行文本

背景

在日常生活中，我们通常会使用特定格式的字符串记录一些信息。在后续的整理过程中，可能希望将这些信息的特定字段解析出来，这种情况下可以使用 Python 中的字符串令牌解析方法。
比如对于以下文本，是具有一定特殊格式规律的。

比如，字符串前 8 位代表了日期，后续跟着中文冒号，一个花括号描述了资金流向为 支出，紧跟着中文逗号，再后者是中括号包含的账户名称 支付宝，以及资金数额，可能还会有备注信息等。
比较朴素的传统方法，可以按照字符串规律截取前几位，再根据特定字符分隔字符串获取结果数组的第几个元素等。但这种方法的弊端在于多样性过于丰富，对于不同的字符串格式都需要艰难地寻找规律，且匹配过程对于异常情况的处理能力较弱。

这里介绍一种新方法，可以使用 字符串令牌解析 方法，可以将想其从左至右将其解析为一个令牌流，再从令牌流中提取想要的字段。

基本思想就是通过一个 (?P<[field_name]>[pattern]) 令牌匹配对应的字段，其中 [field_name] 描述了被匹配内容的字段名称，在匹配结果中可以直接查询，而 [pattern] 则是正则表达式描述匹配规则，比如上例中一个或多个数字的匹配规则可以使用 \d+ 来描述。
将多个令牌拼接在一起，对于固定的字符则可以直接拼在字符串中，这样就能解析出一段匹配文本中的各个域的值。
