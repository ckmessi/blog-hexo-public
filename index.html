添加一个新的规则

使用 iptables -A INPUT -p tcp --dport {port} -j ACCEPT 可以追加一条规则。

···

根据两个点绘制特定宽度的矩形框

背景

在项目中遇到一个特殊的需求，客户希望能够用鼠标拖动的方式点两个点，然后根据这两个点绘制出特定宽度的矩形。
如下图所示：

听起来还挺像一回事的，不过貌似一时半会儿没找着现成的库可以实现这个逻辑，决定自己利用小学数学知识写一个。

解决方案

我们可以把这个过程拆解成三步：
1）根据两点确定一条直线；
2）过这两点向上述直线的垂直方向延伸出目标宽度（设为W）一半的距离；
3）得到延伸后的各两个点共4个点，根据这4个点绘制矩形。

在上述求解过程中，我们知道的输入信息只有两个点的坐标 P1(x1, y1) 和 P2(x2, y2)，要得到另外4个点的坐标。
如果使用解析几何的方式，去计算斜率之类的，感觉会比较复杂，还要考虑到平行于坐标系的情况。
这里我使用的是相似三角形原理去计算：

在不平行或垂直于坐标轴的情况下，P1和P2两点间的连线一定会与两点所在的坐标轴平行线形成直角三角形。
而由于目标是一个矩形，延伸出来的点（如图示点 r1 ）和原始点（如图示点 P1 ）一定也会形成类型的直角三角形。
两个三角形相似，根据相似三角形定理，可知对应的三条边所成比例相等。
参考图中所示的逻辑，可以通过已知参数，计算出 r1 点的坐标 (x_r1, y_r1）。
类似的，也可以求出另外3个点。

根据两个点绘制特定宽度的矩形框，可以使用相似三角形原理，计算出对应矩形的四个点坐标。
